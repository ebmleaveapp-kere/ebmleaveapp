📌 Important Setup Instructions for Vercel Deployment:

  1. Add Authorized Redirect URI in Google Cloud Console:
    → Go to Google Cloud Console → APIs & Services → Credentials
    → Edit your OAuth 2.0 Client ID
    → Add this URI:
  2. Add Authorized JavaScript Origins:
    → Add your Vercel domain:
    → Also add: http://localhost:3000 (for local testing)
  3. Enable Google Sheets API:
    → Go to APIs & Services → Library
    → Search for "Google Sheets API" and enable it
  4. Share your Google Sheet:
    → Share the sheet with your email address used in Google Cloud
    → Give "Editor" permissions
